# git workflow 规范
## 1 概念
+ 主仓库：服务端仓库，团队共用一个，包含 master 分支和各版本分支
+ 个人仓库：服务端仓库，每个用户拥有一个，包含与主仓库同步的 master 分支和各feature分支
+ 本地仓库：用户开发机仓库，日常开发仓库，代码往个人仓库同步
